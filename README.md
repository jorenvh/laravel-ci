# laravel-ci: A Docker image for Continuous Integration

Based on Debian 10 (buster)

## Features
+ PHP 7.4
+ Composer
+ Node.js 18
+ npm 8
+ Yarn
+ Supports [Laravel Dusk](https://laravel.com/docs/master/dusk) (Chromium 90)
